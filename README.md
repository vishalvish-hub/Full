AngularJS Viva (Basic)
1. What is AngularJS?

AngularJS is a JavaScript framework used to build dynamic web pages using data binding and MVC concepts.

2. What is ng-model?

ng-model binds HTML input fields to AngularJS scope variables.

3. What is ng-repeat?

It repeats HTML elements for each item in an array (like a loop).

4. What is $scope?

$scope is an object that connects the controller and the view.

5. What does ng-click do?

It runs a function when a button or element is clicked.

6. Purpose of ng-init?

Used to initialize variables at the start of the page.

7. What is ng-bind?

It displays data just like {{ }}, but prevents flickering.

Two-way binding connects input and variable both ways using [(ngModel)].

A lifecycle hook that runs when the component starts.

13. What is a Service in Angular?

A service is used to write reusable code like calling APIs.
